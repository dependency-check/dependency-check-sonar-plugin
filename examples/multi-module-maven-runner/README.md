Example Multi-Module Maven Project
=====================================

Integrates [Dependency-Check] analysis and reporting into SonarQube v6.7 or higher in a multi-module Maven project.
 
Usage
-------------------

```
mvn clean package dependency-check:check
mvn sonar:sonar
```
